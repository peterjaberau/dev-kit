---
nav: Colors
title: CssVar
description: The CssVar component is a memoized component used to display a color scale. It receives props containing the name of the scale, the color scale items, and a number representing the index of the mid-highlight. The component renders a set of ScaleRow components, each representing a different part of the color scale.
---

## Primary

```tsx
import { CssVar, primary } from '@devkit/ui/color';

export default () => <CssVar name="primary" scale={primary} midHighLight={9} />;
```
